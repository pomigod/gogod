================================================================================
쿼리 목록
================================================================================

1. EQUI JOIN (동등 조인)
   - 가장 기본적인 조인
   - WHERE 절에서 = 조건 사용

2. INNER JOIN (ISO 표준)
   - EQUI JOIN과 동일한 결과
   - ANSI 표준 문법

3. INTERSECT (교집합)
   - 두 SELECT 결과의 공통 부분

4. OUTER JOIN
   - LEFT OUTER JOIN: 왼쪽 테이블 모두 포함
   - RIGHT OUTER JOIN: 오른쪽 테이블 모두 포함

5. CROSS JOIN (카테시안 곱)
   - 모든 조합 생성
   - 주의: 결과 행 수 = (테이블1 행 수) × (테이블2 행 수)

6. UNION / UNION ALL
   - UNION: 중복 제거
   - UNION ALL: 중복 포함

7. MINUS (차집합)
   - 첫 번째 SELECT에는 있고 두 번째 SELECT에는 없는 데이터

8. CONNECT BY (계층형 쿼리)
   - Oracle 전용 계층 구조 조회
   - NEXTEMP 테이블 필요


================================================================================
오류 해결
================================================================================

Q: "table or view does not exist" 오류
A: EMP, DEPT 테이블이 없습니다.
   - SCOTT 스키마에 접속했는지 확인
   - 테이블 생성 필요하면 DBA에게 문의

Q: "NEXTEMP: invalid identifier" 오류
A: NEXTEMP 테이블이 없습니다.
   - CONNECT BY 쿼리는 건너뛰어도 됨
   - 계층형 쿼리 테스트가 필요하면 테이블 생성 필요

Q: "insufficient privileges" 오류
A: 권한이 부족합니다.
   - DBA에게 SELECT 권한 요청
   - 또는 다른 스키마로 접속
