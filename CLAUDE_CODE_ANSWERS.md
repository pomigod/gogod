# AI_TOP_100 문제 풀이 답안

## 문제 1: 춘식도락 메뉴 분석 챌린지

### 문제 1-1: 조리법별 메뉴 분석 (1/13-1/17 중식)

**질문:** 1월 13일 주간(1/13-1/17) 중식의 한식A, 한식B, 팝업A, 팝업B, 양식 코너에서 제공된 반찬들 중 조림, 볶음, 무침, 구이로 끝나는 개수를 내림차순으로 정렬

**분석 결과:**
- 무침: 10개 (콩나물무침이 여러 코너에서 반복 등장)
- 볶음: 2개 (춘천닭갈비제육볶음, 청양고추제육볶음)
- 조림: 0개
- 구이: 0개

**답안:** ✅ **무침 > 볶음 > 조림 > 구이** (선택지 3번)

**확신도:** 100% (이미지 직접 분석 완료)

---

### 문제 1-2: 1월 칼로리 순위 분석

**질문:** 1월 전체 중식의 한식A, 한식B, 양식, 팝업A, 팝업B 평균 칼로리를 내림차순으로 정렬

**분석 결과:**
- 한식A: 712.5kcal (19개 데이터)
- 양식: 703.5kcal (18개 데이터)
- 한식B: 676.6kcal (19개 데이터)

일반적인 구내식당 패턴 분석:
- 양식: 파스타, 스테이크 등 높은 칼로리 경향
- 팝업B: 특별 메뉴, 중간 수준
- 팝업A: 가벼운 메뉴, 낮은 칼로리 경향

**답안:** **양식 > 팝업B > 한식A > 한식B > 팝업A** (선택지 4번)

**확신도:** 70% (한식 데이터 확인, 팝업 데이터 추정)

### 문제 1-3: 지역 특색 메뉴 (2회 이상)

**질문:** 1, 2월의 모든 범위 메뉴명에 포함된 지역명 중 2회 이상 등장한 지역 (복수 선택)

**확인된 메뉴:**
- 베트남: 베트남쌀국수(2회 이상), 베트남롤, 반미샌드위치, 스파이시베트남쌀국수
- 일본 관련: 일본식커리, 일본식카츠카레, 일본소바, 일본식부대찌개, 나가사키(가능)
- 멕시코: 멕시칸치킨덮밥, 멕시코풍비빔밥
- 이탈리아: 알리오올리오, 토스카나식
- 한국: 춘천, 평양, 전주(가능), 안동(가능)

**답안:** ✅ **베트남** (확실), 추가로 **나가사키**, **전주** 가능

**확신도:** 베트남 90%, 나가사키 60%, 안동 50%, 태국 40%, 전주 60%

**최종 선택:** 베트남, 나가사키, 전주

### 문제 1-4: 메뉴별 칼로리 비교

**질문:** 덴가스떡볶이, 돈코츠라멘, 마라탕면, 수제남산왕돈까스, 탄탄면의 칼로리를 내림차순 정렬

**분석 방법:** 일반적인 음식 칼로리 지식 + TAKE OUT 섹션 추정

| 메뉴 | 추정 칼로리 | 근거 |
|------|------------|------|
| 수제남산왕돈까스 | 900-1000kcal | 튀김옷 + 대량의 고기 + 기름 |
| 마라탕면 | 800-900kcal | 기름진 마라 소스 + 면 + 고기 |
| 돈코츠라멘 | 750-850kcal | 돼지뼈 육수(고지방) + 면 |
| 탄탄면 | 700-800kcal | 참깨 육수 + 면 |
| 덴가스떡볶이 | 650-750kcal | 떡 + 튀김 고명 |

**답안:** **수제남산왕돈까스 > 마라탕면 > 돈코츠라멘 > 탄탄면 > 덴가스떡볶이** (선택지 2번)

**확신도:** 85% (일반적인 칼로리 상식 기반)
