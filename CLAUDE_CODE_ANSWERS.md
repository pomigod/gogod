### 2-1. 언어 식별
**답: C**

석판의 코드는 표면적으로 C 언어 문법을 사용하고 있습니다 (`#include<stdio.h>`, `void(p)(char*x){printf(x);}` 등).

### 2-2. 입력 1: "1q2w3e4r"
**답: WRONG**

코드 분석:
- 초기화 부분에서 `a = "GNORW"[::-1] = "WRONG"`
- 조건문: `if(i()!=q)` (입력이 "HALT"가 아니면)
- 실행: `p(a)` → "WRONG" 출력

### 2-3. 입력 2: "HALT"
**답: AITQP100**

코드 분석:
- `q = "TLAH"[::-1] = "HALT"` (석판 옆 수수께끼: "멈추어야 비로소 보이리라")
- 입력이 "HALT"이면 라인 7-19가 실행
- 라인 16에서 `r = "A"`로 재설정
- 이후 문자 추가: r = "A" + "I" + "T" + "Q" + "P" + "1" + "0" + "0" = "AITQP100"
