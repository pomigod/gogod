# Daily Harmony - 일상 관리 서비스

## 프로젝트 개요

**Daily Harmony**는 바쁜 현대인들을 위한 올인원 일상 관리 웹 애플리케이션입니다.
할 일 관리, 습관 트래커, 간단한 가계부 기능을 하나의 직관적인 인터페이스로 제공하여
사용자가 자신의 일상을 체계적으로 관리할 수 있도록 돕습니다.

---

## 주요 기능

### 1. AI 챗봇 수강신청 도우미
- **수강신청 내역 분석**: 사용자가 수강신청 과목 목록을 입력하면 자동으로 파싱
- **시간표 자동 생성**: AI가 최적의 시간표 조합을 추천
- **시간 충돌 감지**: 겹치는 수업 시간을 자동으로 확인하고 경고
- **대화형 인터페이스**: 자연어로 대화하며 시간표 조정 가능

### 2. 할 일 관리 (Todo List)
- 할 일 추가/삭제/완료 표시
- 우선순위 설정
- 마감일 관리
- 카테고리별 분류

### 3. 습관 트래커 (Habit Tracker)
- 매일 체크할 습관 등록
- 달성률 시각화
- 연속 달성 일수 표시
- 동기부여 메시지

### 4. 간단한 가계부
- 수입/지출 기록
- 카테고리별 통계
- 월별 요약
- 간단한 차트 시각화

---

## 기술 스택

### Backend
- **Python Flask**: 웹 서버 프레임워크
- **AWS Bedrock (Claude)**: AI 챗봇 엔진
- **SQLite**: 간단한 데이터 저장

### Frontend
- **HTML5/CSS3**: 기본 구조 및 스타일
- **JavaScript (Vanilla)**: 동적 상호작용
- **Bootstrap 5**: 반응형 UI 프레임워크

```
사용자 브라우저
```

## 수강신청 도우미 사용 예시

**사용자 입력:**
```
컴퓨터과학개론 (월/수 10:00-11:30)
자료구조 (화/목 13:00-14:30)
영어회화 (월/수 13:00-14:00)
선형대수학 (화/목 10:00-12:00)
```

**챗봇 응답:**
```
✅ 입력하신 4개 과목을 분석했습니다!

📊 시간표 조합:
월요일: 10:00-11:30 컴퓨터과학개론, 13:00-14:00 영어회화
화요일: 10:00-12:00 선형대수학, 13:00-14:30 자료구조
수요일: 10:00-11:30 컴퓨터과학개론, 13:00-14:00 영어회화
목요일: 10:00-12:00 선형대수학, 13:00-14:30 자료구조

⚠️ 시간 충돌 없음!
✨ 균형 잡힌 시간표입니다.
```
