## 💾 로컬에 다운로드해서 보기

### 방법 1: 전체 프로젝트 다운로드 (ZIP)

1. GitHub 페이지에서 **초록색 "Code" 버튼** 클릭
2. **"Download ZIP"** 클릭
3. 다운로드된 ZIP 파일 압축 해제
4. 폴더 안의 `.md` 파일들을 텍스트 에디터로 열기

### 방법 2: Git Clone (개발자용)

터미널/명령 프롬프트에서:

```bash
git clone https://github.com/pomigod/gogod.git
cd gogod
git checkout claude/daily-harmony-final-submission-01PGznatAZW6ssXquA76bouv
```

그 다음 파일 탐색기에서 `.md` 파일을 열어보세요!

## 🎬 단계별 따라하기 예시

### Step 1: 가이드 열기

**옵션 A - GitHub에서 바로 보기:**
1. 브라우저 열기
2. 주소창에 붙여넣기:
   ```
   https://github.com/pomigod/gogod/blob/claude/daily-harmony-final-submission-01PGznatAZW6ssXquA76bouv/STEP_BY_STEP_GUIDE.md
   ```
3. Enter!

**옵션 B - 다운로드해서 보기:**
1. ZIP 다운로드
2. 압축 해제
3. `STEP_BY_STEP_GUIDE.md` 파일 더블클릭

### Step 2: 두 개의 창 띄우기

**왼쪽 화면:** GitHub 가이드 (또는 다운로드한 파일)
**오른쪽 화면:** AWS Console

이렇게 나란히 놓고 따라하세요!

### Step 3: 복사-붙여넣기

가이드에 나오는 명령어나 설정값을:
1. 드래그해서 선택
2. Ctrl+C (또는 Cmd+C)로 복사
3. AWS Console이나 터미널에 Ctrl+V로 붙여넣기

### Step 4: 천천히 따라하기

급하게 하지 마세요!
- 각 단계마다 스크린샷처럼 설명되어 있습니다
- 체크박스(☐)를 보면서 하나씩 확인하세요
- 막히면 "문제 해결" 섹션을 보세요

## 💡 Markdown 파일(.md) 보는 프로그램

### Windows
- 메모장 (기본 - 간단히 볼 때)
- Notepad++ (무료 - 추천!)
- VS Code (개발자용)

### Mac
- TextEdit (기본)
- VS Code (개발자용)
- Typora (유료 - 예쁘게 보임)

### 온라인 (가장 쉬움!)
- GitHub에서 바로 보기 (추천!)
- 별도 프로그램 설치 불필요

## 📱 모바일에서 보기

### 스마트폰/태블릿으로도 가능!

- 컴퓨터로 작업하면 편합니다!
