방법 3) 직접 메모장 열기
   notepad CLAUDE_CODE_ANSWERS.md

방법 4) PowerShell에서 열기

   cat CLAUDE_CODE_ANSWERS.md

방법 2) 페이지 단위로 보기
   less CLAUDE_CODE_ANSWERS.md

방법 3) 텍스트 에디터로 열기
   nano CLAUDE_CODE_ANSWERS.md
