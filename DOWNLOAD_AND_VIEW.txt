================================================================================
Windows PowerShell - 복사해서 실행하세요!
================================================================================

irm https://raw.githubusercontent.com/pomigod/gogod/claude/battle-simulation-problems-017FMJ2a6Fqxv2XDnc5DGYjn/problem4_answers.txt -OutFile problem4_answers.txt; notepad problem4_answers.txt


또는 (짧은 버전):

iwr https://raw.githubusercontent.com/pomigod/gogod/claude/battle-simulation-problems-017FMJ2a6Fqxv2XDnc5DGYjn/problem4_answers.txt -OutFile problem4_answers.txt; notepad problem4_answers.txt


================================================================================
Windows CMD (명령 프롬프트) - 복사해서 실행하세요!
================================================================================

curl -L https://raw.githubusercontent.com/pomigod/gogod/claude/battle-simulation-problems-017FMJ2a6Fqxv2XDnc5DGYjn/problem4_answers.txt -o problem4_answers.txt && notepad problem4_answers.txt


또는 PowerShell 사용:

powershell -Command "iwr https://raw.githubusercontent.com/pomigod/gogod/claude/battle-simulation-problems-017FMJ2a6Fqxv2XDnc5DGYjn/problem4_answers.txt -OutFile problem4_answers.txt; notepad problem4_answers.txt"


================================================================================
다른 파일들도 다운로드하려면:
================================================================================

PowerShell:
----------
# 전체 답안 문서 (마크다운)
irm https://raw.githubusercontent.com/pomigod/gogod/claude/battle-simulation-problems-017FMJ2a6Fqxv2XDnc5DGYjn/CLAUDE_CODE_ANSWERS.md -OutFile CLAUDE_CODE_ANSWERS.md; notepad CLAUDE_CODE_ANSWERS.md

# 예측 결과 (JSON)
irm https://raw.githubusercontent.com/pomigod/gogod/claude/battle-simulation-problems-017FMJ2a6Fqxv2XDnc5DGYjn/battle_predictions.json -OutFile battle_predictions.json; notepad battle_predictions.json


CMD:
----
# 전체 답안 문서
curl -L https://raw.githubusercontent.com/pomigod/gogod/claude/battle-simulation-problems-017FMJ2a6Fqxv2XDnc5DGYjn/CLAUDE_CODE_ANSWERS.md -o CLAUDE_CODE_ANSWERS.md && notepad CLAUDE_CODE_ANSWERS.md

# 예측 결과
curl -L https://raw.githubusercontent.com/pomigod/gogod/claude/battle-simulation-problems-017FMJ2a6Fqxv2XDnc5DGYjn/battle_predictions.json -o battle_predictions.json && notepad battle_predictions.json


================================================================================
사용 방법:
================================================================================

1. 위의 명령어 중 하나를 복사
2. PowerShell 또는 CMD를 열기
3. 붙여넣기 후 Enter
4. 자동으로 파일이 다운로드되고 메모장이 열립니다!

주의: 인터넷 연결이 필요합니다.
