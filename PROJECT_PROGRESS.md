## 🚀 배포 가이드

### 사전 요구사항
- AWS 계정 (kyuniv-pj-06)
- EC2 인스턴스 (t3.small 권장)
- IAM 인스턴스 프로필: `SafeInstanceProfileForUser-kyuniv-pj-06`

### 배포 단계

#### 1. EC2 인스턴스 생성
1. AWS Console 로그인 (버지니아 북부 리전)
2. EC2 > 인스턴스 시작
3. AMI: Amazon Linux 2023 선택
4. 인스턴스 타입: t3.small 선택
5. **중요**: 인스턴스 프로필 `SafeInstanceProfileForUser-kyuniv-pj-06` 연결
6. 보안 그룹: 포트 5000 인바운드 허용
7. 인스턴스 시작

#### 2. 프로젝트 배포
```bash
# EC2 인스턴스 접속
ssh -i your-key.pem ec2-user@<EC2-PUBLIC-IP>

# 프로젝트 클론 (또는 파일 업로드)
git clone <repository-url>
cd ai-learning-assistant

# 배포 스크립트 실행
chmod +x deploy.sh
./deploy.sh
```

#### 3. 서비스 확인
```bash
# 서비스 상태 확인
sudo systemctl status ai-assistant

# 로그 확인
sudo journalctl -u ai-assistant -f

# 웹 브라우저에서 접속
# http://<EC2-PUBLIC-IP>:5000
```
