## DeepRacer 보상 함수 (Reward Functions)

이 문서는 AWS DeepRacer 챌린지를 위해 설계된 다섯 가지 보상 함수를 설명합니다.

### 1. 기본 모델 (`1_baseline.py`)
설명: 가장 기본적인 함수입니다. 트랙을 이탈하거나 장애물과 충돌할 경우<br>
매우 낮은 보상을 주고, 트랙 위를 주행하면 기본 보상을 부여하여 완주 자체에 집중하도록 설계되었습니다.

### 2. Waypoints 추적 모델 (`2_waypoint_tracking.py`)
설명: Waypoints를 기반으로 차량의 주행 방향 정확도를 보상에 추가했습니다.<br>
차량의 헤딩이 트랙의 진행 방향과 일치할수록 높은 보상을 주어 안정적인 코너링과 주행을 유도합니다.

### 3. 회피 최적화 모델 (`3_avoidance_basic.py`)
설명: 장애물 회피에 가장 집중하는 모델입니다. 가장 가까운 장애물의 위치(왼쪽/오른쪽)를 파악하고,<br>
차량이 장애물의 반대편 차선으로 주행할 경우 큰 보상을 주어 충돌을 피하도록 직접적으로 유도합니다.

### 4. 대회 최고 기록 코드 1 (`4_final_submission.py`)
설명: 트랙을 3개 구간으로 나누어 직선에서는 과감하게 가속하고 커브에서는 조향각에 따라 보너스를 주도록 짰습니다.<br>
장애물이 1m 이내로 가까워지면 반대 차선으로 확실히 피했을 때만 <br> 
높은 점수를 줘서, 멈추지 않고 능동적으로 피해서 주행하도록 만들었습니다.

### 5. 대회 최고 기록 코드 2 (`5_championship_final.py`)
설명: 대회 최종 제출용으로 안정성과 속도 제어에 가장 공을 들여 튜닝한 버전입니다.<br>
급커브 구간에서 로그 함수를 적용해 안정적으로 진입하게 했고,<br>
차량 헤딩이 트랙 진행 방향과 일치할수록 가중치를 곱해줘서 고속 주행 중에도 차체가 흔들리지 않고 부드럽게 나갑니다.
